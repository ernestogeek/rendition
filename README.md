## Rendition
Create modern web app clean architecture with ReactJS Typescipt + MongoDB & ViteJS + React



## Technologies
- Crawl data
  - Using Python + Scrapy to crawl books data from other website
- Backend
  - NodeJs/NestJs TypeScript
  - MongoDB/Mongoose
  - Cloudinary for uploading images
  - Jest (testing)
- Front-end
  - ReactJS Typescript
  - ViteJS (really fast)
  - Redux + Redux Thunk
  - SCSS
  - Cypress (testing)
